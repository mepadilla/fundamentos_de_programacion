Parte 1: Entorno de Desarrollo (IDE) y Proceso de Creación

Para comenzar a programar en Python, es necesario utilizar un Entorno de Desarrollo Integrado (IDE) o un entorno interactivo (IDLE) 1.

  • Python viene con su propio IDLE conformado por una consola y un editor de texto 5.
  • Sin embargo, en esta asignatura utilizaremos Thonny como IDE para el desarrollo de los programas 6. Thonny es fácil de usar, viene con Python integrado, resalta errores de sintaxis (como comillas abiertas o falta de paréntesis) y permite ejecutar programas paso a paso para observar cómo cambian los valores de las variables (depuración) 7, 8.

El proceso para la creación de un programa consiste en:

  1. Edición del código fuente 9.
  2. Ejecución mediante el intérprete, el cual traduce y ejecuta línea a línea 10.
  3. Verificación y depuración (corregir errores de sintaxis o de lógica) 9.

Parte 2: Designación de Identificadores

El identificador es el nombre que se le asigna a una variable o constante en el código 2. Para crearlo se deben cumplir las siguientes reglas 2, 11:

  1. Debe comenzar por una letra (preferiblemente en minúscula).
  2. No puede contener caracteres especiales ni espacios, excepto el guion bajo (_).
  3. Python distingue entre mayúsculas y minúsculas (por ejemplo, A y a son variables distintas).
  4. No se pueden usar palabras reservadas del lenguaje (como if, for, while, import, True, etc.) 12.

Parte 3: Instrucciones Básicas de Entrada y Salida

Para que nuestro programa sea interactivo, debe poder recibir datos del usuario y mostrar resultados 1.

  • Salida estándar (print): Muestra información en la pantalla. Ejemplo: print('Hola mundo') 14.
  • Entrada de datos (input): Detiene la ejecución del programa para que el usuario introduzca texto mediante el teclado. Se debe usar junto con funciones de conversión de tipos de datos, como int() para enteros o float() para reales, si se van a realizar cálculos matemáticos 15, 16. Ejemplo: n1 = float(input("Introduce el primer número: ")) 17.

Parte 4: Operadores y Expresiones Aritméticas

Las expresiones aritméticas se construyen utilizando operadores que devuelven un resultado numérico 18.Los operadores aritméticos en Python son 3, 19:

  • + (Suma)
  • - (Resta o Negación)
  • * (Multiplicación)
  • / (División real)
  • // (División entera: devuelve la parte entera del cociente)
  • % (Módulo: devuelve el resto de la división)
  • ** (Exponente o Potencia)

Jerarquía (Prioridad) de Operadores 20:Al evaluar expresiones, Python sigue un orden estricto de precedencia matemático:

  1. Potenciación (**)
  2. Negación (-)
  3. Multiplicación y división real (*, /)
  4. División entera (//)
  5. Resto de división (%)
  6. Suma y resta (+, -)

Nota: Se pueden utilizar paréntesis () para alterar esta prioridad y agrupar operaciones 21.
