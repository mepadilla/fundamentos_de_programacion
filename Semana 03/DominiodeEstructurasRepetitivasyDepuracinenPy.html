Parte 4: Herramientas y Técnicas de Programación

Para resolver problemas complejos dentro de los ciclos, utilizamos variables con roles específicos 10:

  1. Acumuladores: Variable para ir guardando el resultado parcial de una operación (como sumas o multiplicaciones). Debe inicializarse antes del ciclo con el elemento neutro (0 para sumas, 1 para productos) 11.
  2. Contadores: Un tipo especial de acumulador que se incrementa en un valor constante (generalmente 1) cada vez que ocurre un evento 12, 13.
  3. Banderas (Indicadores): Una variable lógica (True o False) que sirve para recordar si un evento ha sucedido o no durante las iteraciones 13.
